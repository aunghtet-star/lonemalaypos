## ğŸ“ Understanding the Interface

### Key Principles

1. **Visual Hierarchy**: Most important = Most visible
2. **Color Coding**: Consistent meaning throughout
3. **Touch Friendly**: Large targets for mobile
4. **Fast Access**: Most used features most accessible
5. **Clear Feedback**: Every action has visual response

### Mental Model

Think of the interface as:
- **Header** = Navigation & filters
- **Quick Bar** = Express train (one-click switching)
- **Menu Area** = Shopping area
- **Cart** = Current workspace
- **Modals** = Deep dive / special actions
